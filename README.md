# OIBSIP Web Development and Designing - Task 1

## Project Overview

This repository contains **Task 1 (Landing Page)** for the Oasis Infobyte Web Development and Designing Internship.

The landing page is a complete, responsive, and modern static webpage built using only:

- HTML5
- CSS3

Theme used in this project: **CloudSync Pro - AI-Powered Cloud Storage and Collaboration**.

## Objective of the Task

The primary goal of this task was to design and develop a professional landing page that demonstrates:

- Clean semantic HTML structure
- Modern UI/UX styling
- Responsive design for desktop, tablet, and mobile
- Visual hierarchy and section-based storytelling
- Reusable and maintainable CSS architecture

## Features Implemented

The landing page includes the following sections:

1. Sticky navigation header
2. Hero section with call-to-action buttons
3. Features section
4. About section with stats
5. Benefits section
6. Testimonials section
7. Pricing section (including featured plan)
8. Call-to-action banner
9. Footer with product/company/social links

Additional UI features:

- Smooth scrolling
- CSS gradients and shadows
- Card hover interactions
- Animated hero elements (floating cards/orb)
- Consistent design tokens via CSS variables

## Tech Stack

- **HTML5** for page structure
- **CSS3** for layout, styling, responsiveness, and animations
- No JavaScript framework or external UI library used

## Process Followed

This is the implementation process followed to complete the task:

1. Planned a product-style landing page concept and section flow.
2. Created semantic HTML structure in `index.html`.
3. Added reusable CSS variables (colors, spacing, radius, shadows, typography).
4. Built each section as modular blocks for easier maintenance.
5. Added visual polish using gradients, cards, and hover states.
6. Implemented responsive breakpoints for tablet and mobile.
7. Tested in local browser using a static HTTP server.
8. Prepared project documentation and pushed to GitHub.
